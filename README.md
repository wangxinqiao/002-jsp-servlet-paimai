基于B2C的网上拍卖系统主要用于帮助人们应用互联网方便快捷买到自己所中意的商品，并参与到秒杀与竞拍当中。
主要功能包括：

1.前台模块
(1)普通用户登录/注册。
(2)分类查看商品(普通商品与促销商品)
(3)查看商品详细信息
(4)查看秒杀商品
(5)查看竞拍商品
(6)将商品加入购物车
(7)购买，结算功能
(8)留言

2.后台模块
(1)修改密码
(2)商品管理:

-- 编辑/删除

-- 设置/取消促销

(3)秒杀商品：设置/取消秒杀
(4)竞拍商品：设置/取消竞拍
(5)订单管理：查看订单
(6)留言管理：查看/删除留言

项目访问路径:
前台：http://localhost:8080/sale   用户名：admin 密码：123456
后台：http://localhost:8080/sale/user/adminlogin   用户名：admin 密码：123456
